The API tokens can be used for authenticating yourself in the REST API call. See our wiki for more details.
The username associated with each token is your Jenkins username.

Some good practices for keeping your API tokens secure are:
  • Use a different token for each application (if one application is compromised, you do not lose data from other)
  • Regenerate the tokens every 6 months (depending on your context). We put an indicator concerning the age of the token.
  • Protect it like your password, as it allows other people to access Jenkins as you.
