Gruppen (auch: Labels oder Tags) werden benutzt, um mehrere Agent-Knoten in eine logische Gruppe zusammenzufassen. Die Angabe 'regression java6' ordnet beispielsweise einen Knoten den Gruppen 'regression' und 'java6' zu.

Beispiel: Sie haben mehrere Agent-Knoten unter Windows und einen Job, der Windows erfordert.
