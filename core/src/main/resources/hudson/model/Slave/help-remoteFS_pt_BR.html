Um agent precisa ter um diretório dedicado ao Jenkins. Especifique o caminho absoluto deste diretório de trabalho no agent, tal como '/export/home/jenkins'.

Agents não mantém dados importantes (além dos workspaces ativos dos últimos projetos construídos neles), assim você pode configurar o workspace do agent para um diretório temporário. A única desvantagem de fazer isto é que você pode perder o workspace atualizado se o agent for desligado.
